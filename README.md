# Family Visit Attribution Demo

Public static demo for multi-patient visit speech attribution and note generation.

Live site: [family-visit-attribution-demo.pages.dev](https://family-visit-attribution-demo.pages.dev)

## Files

- `index.html`: the interactive speech attribution and clinical note demo
- `assets/family-visit-demo-audio.m4a`: bundled demo audio used by the attribution experience

## Local Preview

Run a simple static server from this folder:

```bash
python3 -m http.server 4173
```

Then open [http://localhost:4173/index.html](http://localhost:4173/index.html).
